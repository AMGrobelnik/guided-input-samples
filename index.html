Guided run setup — ten ways the questions can live in the box

A run normally starts by typing a prompt — that still works exactly as today. This adds an opt-in path beside it: press Guide me and, instead of writing the whole thing, you answer a few quick questions. Three are fixed; once they're answered, three more are generated from your answers. When you're done, the six answers are stitched into a single prompt and sent to start the run — identical to one you'd type by hand. Below are ten live treatments for how that flow can live in the input box — click through any of them.
